Pertanyaan Umum

Bagaimana cara berlangganan?

Klik "Pilih Paket" di kartu paket pilihan Anda, isi formulir, dan tim kami akan menghubungi dalam 1x24 jam.

Apa itu FUP?

FUP (Fair Usage Policy) adalah batas pemakaian wajar. Setelah mencapai FUP, kecepatan akan disesuaikan untuk menjaga kualitas jaringan.

Bisa upgrade paket?

Tentu! Anda bisa upgrade paket kapan saja. Perubahan berlaku di bulan berikutnya tanpa biaya tambahan.

Apakah paket SOHO bisa untuk rumahan?

Bisa! Paket SOHO cocok untuk rumahan dengan kebutuhan tinggi seperti content creator, streamer, atau smart home advanced.

Berapa lama instalasi?

Instalasi standar 2-4 jam. Untuk paket SOHO, kami lakukan survey lokasi terlebih dahulu (1-2 hari) untuk optimasi jaringan.

Apakah ada kontrak?

Tidak ada kontrak jangka panjang. Anda bisa berlangganan bulanan dan berhenti kapan saja. Untuk SOHO minimal 3 bulan.

Masih Bingung Memilih Paket?

Tim kami siap membantu merekomendasikan paket terbaik sesuai kebutuhan Anda.
