## Mengapa Keamanan WiFi Penting?

Tahukah Anda bahwa jaringan WiFi yang tidak aman bisa:
- Dicuri bandwidth-nya oleh tetangga
- Diretas untuk mencuri data pribadi
- Digunakan untuk aktivitas ilegal
- Memperlambat koneksi internet Anda

## 7 Langkah Mengamankan WiFi Rumah

### 1. Ganti Password Default Router

Jangan gunakan password bawaan pabrik! Buat password yang:
- Minimal 12 karakter
- Kombinasi huruf besar, kecil, angka, dan simbol
- Tidak menggunakan informasi pribadi
- Diganti secara berkala

### 2. Gunakan Enkripsi WPA3/WPA2

Pastikan router Anda menggunakan:
- **WPA3** (terbaru dan teraman)
- **WPA2-AES** (minimum)
- Hindari WEP atau WPA-TKIP (sudah usang)

### 3. Sembunyikan SSID

Dengan menyembunyikan nama WiFi:
- Jaringan tidak terlihat oleh orang luar
- Mengurangi risiko peretasan
- Hanya yang tahu nama SSID yang bisa connect

### 4. Aktifkan Firewall

Firewall melindungi dari:
- Serangan malware
- Akses tidak sah
- Port scanning
- Traffic mencurigakan

### 5. Update Firmware Router Secara Rutin

Firmware terbaru memberikan:
- Patch keamanan
- Fitur baru
- Performa lebih baik
- Bug fixes

### 6. Gunakan VPN

VPN dari RGN menyediakan:
- Enkripsi data end-to-end
- Menyembunyikan IP asli
- Akses konten global
- Perlindungan di WiFi publik

### 7. Monitor Perangkat Terhubung

Cek secara berkala:
- Perangkat yang terhubung ke WiFi
- Aktivitas mencurigakan
- Bandwidth usage per perangkat
- Blokir perangkat tidak dikenal
